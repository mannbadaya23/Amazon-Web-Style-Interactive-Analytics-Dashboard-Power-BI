# Amazon Web-Style Interactive Analytics Dashboard | Power BI

Interactive multi-page Power BI dashboard developed to analyze Amazon sales performance, seller activity, regional insights, and product-level trends through advanced business intelligence reporting and modern dashboard interactions.

## Project Overview

This project focuses on building a web-style analytical reporting experience using Power BI with interactive navigation, KPI monitoring, custom tooltips, drill-through analysis, and dynamic product-detail pages.

## Key Features

* KPI Cards & Business Metrics
* Product-Level Sales Analysis
* Custom Tooltip Reporting
* Interactive Navigation Buttons
* Dynamic Product Pages
* Drill-Through Analysis
* State-wise & City-wise Insights
* Seller & Product Performance Tracking
* Responsive Dashboard Layout

## Dashboard Impact & Optimization

* Improved analytical visibility through interactive business intelligence reporting.
* Reduced manual analysis effort by consolidating multiple sales insights into a single reporting dashboard.
* Enhanced user experience using tooltip interactions, slicers, and navigation-based reporting.
* Optimized dashboard layout, visual hierarchy, and reporting accessibility for faster business insights and data exploration.

## Tools & Technologies Used

* Power BI

* Power Query
* Excel

* Business Intelligence
